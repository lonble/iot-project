# iot-project

This is the repo for the IoT Communication project.

# Structure

The different components that make up this project are located in separate directories. Please visit each component's directory and view its **README** file for details.

# Build and Run

First, build the program using the code provided in the `board` directory and flash it onto boards. Then, use boards to collect data. Detailed build steps and usage instructions can be found in the README file within the subdirectory.
